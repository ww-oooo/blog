SQL 四大操作语句

Mat 20, 2017

删除 DELETE

DELETE FORM 表 WHERE

增加 INSERT

INSERT INTO 表(字段列表)  VALUES(值列表)

改 UPDATE

UPDATE 表 SET 字段=值,字段=值,... WHERE

查 SELECT

SELECT * FORM 表 WHERE

子句:

WHERE   条件
ORDER   排序
GROUP   聚类-合并相同

假如现在有一张表,表里分别是不同班级的学生,查询每个班级有几个人:
SELECT 班级,COUNT(班级) FORM 表名 GROUP BY 班级
